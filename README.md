# Python Security Tools

A collection of beginner-friendly Python cybersecurity projects created to build practical skills in security automation, password security, and threat awareness.

## Projects

### Password Strength Analyzer

A Python tool that evaluates a password based on common security requirements:

- At least 12 characters
- Uppercase letter
- Lowercase letter
- Number
- Special character

The tool labels passwords as Weak, Moderate, or Strong and gives suggestions for improvement.

## How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run:

```bash
python password_strength_analyzer.py
